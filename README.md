# Home Energy Intelligence

A home energy monitoring platform designed to collect, store and analyse:

- Solar generation
- Battery state of charge
- House energy consumption
- Grid import/export
- Weather conditions
- Indoor environmental data

## Planned Features

### Data Collection
- GoodWe API integration
- Weather API integration
- Timestamped data logging

### Monitoring
- Historical reporting
- Grafana dashboards
- Environmental monitoring

### AI & Analytics
- Daily AI summaries
- Anomaly detection
- Energy forecasting

## Technology Stack

- Python
- Docker
- SQLite
- Grafana
- Ubuntu Linux
- GoodWe API
